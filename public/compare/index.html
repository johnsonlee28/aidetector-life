How We Test#

We use a standardized test set of 50 texts:

  • 20 human-written (blog posts, essays, emails)
  • 20 generated by ChatGPT-4o
  • 10 generated by Claude 3.5 Sonnet

Each tool is tested blind on the same inputs. We measure true positive rate (AI text correctly identified) and false positive rate (human text incorrectly flagged).
